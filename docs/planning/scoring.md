# 점수 계산 규칙 (Scoring)

> `rules.md`와 짝을 이루는 문서. 획득한 카드 → 점수 → 배수 → 최종 점수 순서로 계산한다.
> 애매한 규칙은 하단 **⚠️ 결정 필요**에 모음. 최종 갱신: 2026-07-07 · 2인 맞고

---

## 1. 기본 점수

### 1.1 광 (光)
| 보유 광 | 점수 | 비고 |
|---|---|---|
| 3광 (비광 미포함) | 3점 | |
| 3광 (비광 포함) | 2점 | 비광이 낀 3광은 감점 |
| 4광 | 4점 | |
| 5광 | 15점 | |

### 1.2 열끗 (10끗)
- 5장부터 1점, 이후 1장당 +1점. (5장=1점, 6장=2점, …)
- **고도리** = 2월 매조 + 8월 기러기 + 10월 사슴 → **5점** (열끗 장수 점수와 별도 합산)
- 9월 국진을 쌍피로 쓸 경우 열끗에서 제외(⚠️ 결정 #1과 연동)

### 1.3 띠
- 5장부터 1점, 이후 1장당 +1점.
- **홍단(1·2·3월) = 3점**, **청단(6·9·10월) = 3점**, **초단(4·5·7월) = 3점** (각각 별도 합산)

### 1.4 피
- **10장부터 1점**, 이후 1장당 +1점. (10장=1점, 11장=2점, …)
- 쌍피는 2장으로 계산(쌍피 채택 시).

---

## 2. 배수 (승자 최종 점수에 곱/가산)

| 이름 | 조건 | 효과 |
|---|---|---|
| 피박 | 승자가 이길 때 상대 피가 5장 이하 | 승자 점수 ×2 |
| 광박 | 승자가 광 점수(3점↑)로 이길 때 상대 광 0장 | 승자 점수 ×2 |
| 멍박(멍텅구리) | 상대 열끗 0장 (2인 적용 여부 ⚠️#2) | ×2 |
| 흔들기 | rules 4.7 | ×2 |
| 폭탄 | rules 4.7 | ×2 |
| 고박 | 고 부른 뒤 상대에게 짐 | 진 사람이 상대 점수만큼 물어줌 |

- 배수는 **곱연산으로 누적**한다. 예: 피박 + 흔들기 = ×4. (누적 방식 ⚠️#3)

---

## 3. 고 / 스톱 점수 처리

기본안(⚠️ 결정 #4 에서 확정):
- 3점 이상에서 고/스톱 선택.
- **1고, 2고**: 최종 점수에 각 **+1점** 가산.
- **3고부터**: 이전까지 점수에 **×2**, 이후 고를 부를 때마다 다시 ×2.
- 예: 기본 3점에서 → 1고(스톱 시 4점) → 2고(5점) → 3고(×2 = 10점) → 4고(×4 = 20점).

> 대안: "매 고마다 +1점", "매 고마다 ×2" 등 앱마다 다름. 기본안은 위 방식.

---

## 4. 계산 순서 (구현 기준)
```
1) 기본 점수 합산 = 광 + 열끗(+고도리) + 띠(+홍/청/초단) + 피
2) 3점 미만이면 스톱 불가 → 판 계속
3) 스톱 시점 점수에 고 가산/배수 적용
4) 박(피박/광박/멍박) 배수 적용
5) 흔들기/폭탄 배수 적용
6) 고박 여부 정산
=> 최종 점수(포인트/판돈 환산은 MVP 이후)
```

## 5. 예시 계산

| # | 승자 획득 | 기본점수 | 상태 | 배수 | 최종 |
|---|---|---|---|---|---|
| 1 | 광3(비광X), 피11장 | 광3 + 피2 = 5점 | 없음 | – | **5점** |
| 2 | 고도리, 피12장, 상대 피 4장 | 고도리5 + 피3 = 8점 | 피박 | ×2 | **16점** |
| 3 | 광3 + 흔들기, 상대 광0 | 광3 = 3점 | 광박+흔들기 | ×2×2 | **12점** |
| 4 | 홍단3 + 피10장, 2고 후 스톱 | 띠3 + 피1 = 4점 | 2고(+2) | – | **6점** |

## 6. 구현 수용 기준 (Acceptance Criteria)
- [ ] 광 3/3(비광)/4/5 점수가 정확하다.
- [ ] 열끗·띠·피의 "N장부터 1점" 임계값이 정확하다.
- [ ] 고도리/홍단/청단/초단 조합이 정확히 판정된다.
- [ ] 피박·광박이 상대 카드 상태에 따라 올바르게 발동한다.
- [ ] 고 가산/배수 규칙이 기본안대로 계산된다.
- [ ] 흔들기·폭탄·박의 배수가 곱연산으로 누적된다.
- [ ] 고박 정산이 승패를 뒤집는 케이스에서 올바르다.
- [ ] 위 예시 계산 4종이 단위 테스트로 재현된다.
